Explicit & Value

Value - W AngularJS jest to prosty obiekt. Może to być ciąg znaków, liczba lub obiekt JS. Obiekty Value najczęsciej stosowane są do konfiguracji, wstrzykiwane do fabryk, serwisów badź kontrolerów

      $scope.objectValue = {{ objectValue}}
      $scope.stringValue = {{ stringValue}}
      $scope.numberValue = {{ numberValue}}
